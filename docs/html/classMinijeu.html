Description détaillée

Mini-jeu jouable pour 2, 3 ou 4 joueurs.

L'implémentation de base utilise un tirage aléatoire. Les mini-jeux graphiques (Tron, Pong…) héritent de cette classe et surchargent les méthodes jouer().
